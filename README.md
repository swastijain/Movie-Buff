# Movie-Buff
Movie Buff is an app that displays all the popular shows on TMDB. The data is being fetched using the TMDB API.

Highlights of the app:
<ul>
<li>Network Handling</li>
<li>Caching API calls</li>
<li>Responsive UI</li>
</ul>
